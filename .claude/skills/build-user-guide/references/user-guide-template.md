## Template Variations

### Quickstart (~1000 words)
- Getting Started (prerequisites + first launch)
- 2-3 core tasks (most common things)
- FAQ (3-5 questions)

### Standard Guide (~2500 words)
- Getting Started
- 4-6 core task sections
- Tips & Tricks
- FAQ

### Comprehensive Reference (~5000+ words)
- Getting Started with key concepts table
- 6-10 sections covering every feature
- Settings/configuration section
- Admin section (if applicable)
- Troubleshooting section
- Full FAQ
- Glossary

---

## Checklist

Before finalizing, verify:
- [ ] Table of Contents links to every section
- [ ] Every section title describes a task or goal (not an internal feature name)
- [ ] Numbered steps for all sequential actions
- [ ] UI elements are bolded consistently: **Save**, **Settings**, etc.
- [ ] Screenshot placeholders inserted where visuals would help
- [ ] Tip and Important blockquotes used for key callouts
- [ ] FAQ covers the 5-10 questions a new user would most likely ask
- [ ] Second person ("you") used throughout
- [ ] Present tense and active voice
- [ ] Humanized Writing Standard followed
- [ ] No internal jargon or feature codenames (use customer-facing terms only)
